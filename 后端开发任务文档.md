# 后端开发任务文档

> 项目：基于 Spring Boot 的新能源汽车充电桩运营管理系统  
> 技术栈：Spring Boot 3.2.x + MyBatis-Plus + MySQL + Spring Security + JWT  
> 状态说明：`未开发` / `开发中` / `未测试` / `已完成`
