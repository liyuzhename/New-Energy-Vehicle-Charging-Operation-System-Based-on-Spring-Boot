## 登录接口说明

```
POST /api/auth/login
Content-Type: application/json

{
  "username": "user01",
  "password": "admin123"
}
```

登录成功后返回 JWT Token，后续请求在 Header 中携带：

```
Authorization: Bearer <token>
```

## 数据库初始化说明

| SQL 文件 | 用途 |
|---------|------|
| `src/main/resources/sql/init.sql` | 建表语句 + 初始管理员账号 |
| `src/main/resources/sql/test_data.sql` | 首次导入测试数据 |
| `src/main/resources/sql/fix_test_data.sql` | 修复旧账号密码 + 补充测试数据 |

**重新初始化命令（Windows CMD）：**

```cmd
mysql -u root -p --default-character-set=utf8mb4 < src\main\resources\sql\init.sql
mysql -u root -p --default-character-set=utf8mb4 < src\main\resources\sql\fix_test_data.sql
```
